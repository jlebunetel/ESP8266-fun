# ESP8266-fun
Having fun with the ESP8266

## software
I use the Arduino IDE 1.6.12 configured with the board model *NodeMCU 1.0 (ESP-12E Module)*.

## hardware
I use a LoLin ESP-12E NodeMcu V3 development board (with a ESP8266) for my experiments.
